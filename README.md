# Minimal Academic GitHub Pages Template

This is a lightweight academic homepage template for GitHub Pages in a clean, minimal style.

## How to use

1. Create a repository named:
   `YOUR-USERNAME.github.io`
2. Upload all files from this template into that repository.
3. Replace the placeholder content in:
   - `index.md`
   - `publications.md`
   - `teaching.md`
   - `projects.md`
   - `cv.md`
   - `_config.yml`
4. Go to **Settings → Pages** in GitHub and enable GitHub Pages from the `main` branch.
5. Your site will appear at:
   `https://YOUR-USERNAME.github.io`

## Files

- `_config.yml` – site metadata
- `_layouts/default.html` – page layout
- `assets/style.css` – styling
- `index.md` – homepage
- `publications.md` – publications page
- `teaching.md` – teaching page
- `projects.md` – projects page
- `cv.md` – CV page

## Customization ideas

- Add a profile photo to the homepage
- Add a dedicated page for students or group members
- Link PDFs, notebooks, and code repositories
- Add a news section or a blog
